128-bit SVE

Find the last (leftmost) 8-bit element from (2) where the corresponding predicate bit in (1) is non-zero, then broadcast the next element to all 8-bit lanes of (4). If the last corresponding predicate bit is non-zero, broadcast the first (rightmost) element from (1) to all lanes of (4). If all corresponding predicate bits are zero, preserve the value from (3).
