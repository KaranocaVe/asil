LD4Q (scalar plus immediate): Contiguous load four-quadword structures to four vectors (immediate index)

128-bit SVE

Load and deinterleave groups of four interleaved 128-bit values from the memory operand (1) into the 128-bit elements of four consecutive registers (2), (3), (4), and (5). If the predicate bit corresponding to an element in (2), (3), (4), and (5) is zero, those four contiguous loads are skipped, and cannot cause a fault, and the elements are set to zero.
