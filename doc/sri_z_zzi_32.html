SRI: Shift right and insert (immediate)

SRI Zd.S, Zn.S, #const (SVE2 (SME
svint32_t svsri[_n_s32](svint32_t op1, svint32_t op2, uint64_t imm3)
svuint32_t svsri[_n_u32](svuint32_t op1, svuint32_t op2, uint64_t imm3)

128-bit SVE

For each 32-bit unsigned integer set (3) to ( (2) >> const ) | ( (1) & ~( 0xFFFFFFFF >> const ) ). This preserves the const most-significant bits from (1), while inserting the ( 32 − const ) most-significant bits from (2) into the least-significant bits of (3). The shift amount is limited to 1 ≤ const ≤ 32.
