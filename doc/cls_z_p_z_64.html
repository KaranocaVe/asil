CLS: Count leading sign bits (predicated)

CLS Zd.D, Pg/M, Zn.D (SVE (SME
svuint64_t svcls[_s64]_m(svuint64_t inactive, svbool_t pg, svint64_t op)

128-bit SVE

For each 64-bit integer set (2) to the number of leading bits that are equal to the sign bit in (1), excluding the sign bit itself. If (1) is zero or 0xFFFFFFFFFFFFFFFF, the result is 63.
